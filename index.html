About

What

A Game Jam is a hackathon for game developers! Create a game individually, or in groups up to 4 within 52 hours.

Why

To collaborate with others and gain a better grasp on developing games!

When

The event will begin December 6th with a kickoff at 5pm and last until December 8th, 12:00AM.

Where

The event will be held online. Participants only have to join a call during the kickoff (Highly Recommended), and the rest of the event can be done with your teams/individually and submitted through itch.io.

Frequently Asked Questions


Do I need to be a part of HDSB?
No! Any student can join the Game Jam online, including those from HCDSB. All students across Ontario are welcome to participate.
Do I need to have prior experience?
No! You don't need any prior experience during this event. We highly encourage learning as you go. (We will also teach you ways to find and solve errors).
Do I need to use any specific programming language or game engine?
No! You're free to use any programming language or game engine you prefer. We recommend Unity, Unreal Engine, or Godot, as these are widely used in the gaming industry and can be valuable for future experience. As long as the game is playable in a web browser via Web Graphics Library (WebGL) or HTML5 and can be uploaded to Itch.io, it's acceptable.
Do I have to develop everything from scratch? (scripts, models, animations)
No! You can use assets from the internet as long as you have the rights to use them. We recommend using free assets from websites like itch.io, opengameart.org, and kenney.nl. Plus, the use of AI is allowed, but make sure your game is copyright free!
